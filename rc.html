About

This project is the creation of a remote control DAF truck, specifically the electric XD model. It is primarily developed by Burnt Clutch, and you can find their YouTube channel here.

Progress

19 Mar 26, Project starts

All of the planning for this project starts at this point, such as materials and electronics. The CAD also begins.

21 Mar 26, CAD completed

The CAD is completed.

23 Mar 26, CAD sent to be printed

All of the CAD files have been sent to be printed.

2 Apr 26, Electronics started

We have started to do the electronics for the DAF, while we wait for the 3D parts to arrive.

3 Apr 26, CAD issues

Issues with the 3D printer not being able to print all of the body at the same time. We sent the body in a top half and a bottom half. Also the Raspberry Pi Pico that we are using blew up and we didn't have spares so we have ordered more!

5 Apr 26, Gofundme launched

Fundraiser launched on Gofundme. You can find the link to it here.
